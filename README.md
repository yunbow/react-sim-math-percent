# パーセントシミュレーター

React 18とTypeScript、Atomic Designパターンで構築されたパーセント（百分率）・割・割分の可視化・相互変換アプリケーションです。

## デモプレイ
https://yunbow.github.io/react-sim-math-percent/demo/

## 主要機能

### 1. 可視化機能
- **10x10グリッド表示**
  - 100マスのグリッドでパーセントを視覚的に表現
  - 基準値に応じてセルの色分けを表示
- **アニメーション演出**
  - パーセント計算の過程をステップバイステップでアニメーション表示
  - 割引・増加の結果を動的に可視化

### 2. 操作モード
- **割引モード**
  - 元の値から指定パーセントを引いた結果を計算・表示
- **増加モード**
  - 元の値に指定パーセントを加えた結果を計算・表示
- **スライダー操作**
  - 基準値とパーセント値をスライダーで直感的に変更可能

### 3. 言葉と数字の変換
- **パーセント表記**
  - 100を基準にした割合（例：20%）
- **割表記**
  - 10を基準にした割合（例：2割）
- **割分表記**
  - 割と分を組み合わせた表記（例：2割5分 = 25%）
- **覚え方ガイド**
  - 1割 = 10%、1分 = 1% の関係を表示

### 4. 計算プロセスの解説表示
- **ステップバイステップ解説**
  - 計算過程を順番に表示
- **アニメーション連動**
  - 解説とグリッドアニメーションが同期

### 5. 練習モード（クイズ）
- **3段階のレベル設定**
  - レベル1: 100を基準にした割合
  - レベル2: 任意の数（80・200など）
  - レベル3: 様々な割合の組み合わせ
- **問題形式**
  - 「○○円の△割引はいくら？」形式で出題
- **ヒント機能**
  - 計算のヒントを表示可能
- **スコア管理**
  - 正解数と総問題数を表示

## 技術スタック

- **React 18** - UIライブラリ
- **TypeScript** - プログラミング言語
- **Storybook 7** - コンポーネント開発・ドキュメント
- **CSS Modules** - スタイリング
- **Vite** - ビルドツール

## プロジェクト構造

```
src/
├── features/                        # 機能別モジュール
│   └── percent/                     # パーセント機能
│       ├── components/              # 機能専用コンポーネント
│       │   ├── PercentGrid/         # パーセント可視化グリッド
│       │   ├── ControlPanel/        # コントロールパネル
│       │   ├── ConversionPanel/     # 変換パネル
│       │   ├── ExplanationPanel/    # 解説パネル
│       │   └── QuizMode/            # クイズモード
│       ├── hooks/                   # カスタムフック
│       │   └── usePercentSimulator.ts
│       ├── utils/                   # ユーティリティ関数
│       │   └── conversion.ts        # 変換計算
│       └── PercentSimulator/        # メインシミュレーター
├── components/                      # 共通UIコンポーネント
│   ├── Button/                      # ボタン
│   ├── Select/                      # セレクトボックス
│   ├── Slider/                      # スライダー
│   └── Input/                       # テキスト入力
├── types/                           # 汎用的な型定義
│   └── index.ts
├── App.tsx                          # メインアプリ
├── main.tsx                         # エントリーポイント
└── theme.css                        # テーマカラー定義
```

## スクリプト

```bash
# セットアップ
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview

# Storybook起動
npm run storybook

# Storybook ビルド
npm run build-storybook
```
